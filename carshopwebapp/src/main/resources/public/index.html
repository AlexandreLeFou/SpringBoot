Welcome to Team's 2 Project Future assignment!

The aim of this site is to present a CRUD application developed using Java Spring. The aim of this project was to deeply understand concers regarding the 3-tier Architecture used by this MVC framework.

User/Owner Site

Insert (User information)

This page shows all the data regarding the data entry of a User's information (update/delete functionality available for each line).

Display (Users' accounts)

This page shows all the data regarding the Users'/Owners' accounts information.

Update (User information)

This page shows all the data regarding updating the entry of a User's information(e.g. id=1).

Repair Site

Insert (Repair information)

This page shows all the data regarding the data entry of a car repair (update/delete functionality available for each line).

Display (Repair data)

This page shows all the data regarding all available car rapair data.

Update (Repair entry)

This page shows all the data regarding updating a single entry of a User's information(e.g. id=5).
